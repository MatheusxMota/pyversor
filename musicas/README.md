# Músicas de Fundo

Coloque aqui seus arquivos de áudio (formato .mp3 ou .wav) para serem usados como trilha sonora nos vídeos gerados.

Certifique-se de que as faixas tenham licença de uso livre para fins comerciais (ex: YouTube Audio Library, Pixabay Music).

O sistema selecionará automaticamente uma música desta pasta para mixar com a narração.
